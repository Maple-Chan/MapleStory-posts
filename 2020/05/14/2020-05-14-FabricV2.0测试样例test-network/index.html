隐私数据

隐私数据集合定义：

一个集合定义包含了一个或者多个集合，每个集合具有一个策略列出了在集合中的所有组织，还包括用来控制在背书阶段私有数据的传播所使用的属性，另外还有一个可选项来决定数据是否会被删除。

在2.0版本，隐私数据集合将由channel成员进行定义审批条件，当链码定义被提交到通道时部署隐私数据集合。
